# Lotto Number Generator Blueprint

## Overview
A high-quality, professional web application that generates 6 unique random numbers between 1 and 45, optimized for Google AdSense approval.

## Project Details
- **Tech Stack:** HTML5, CSS3 (Modern Baseline), Vanilla JavaScript (ES Modules).
- **Design Philosophy:** Clean, responsive, and user-friendly with support for both Light and Dark themes.
- **Key Features:**
    - Unique number generation (1-45).
    - Responsive layout.
    - Theme switching (Light/Dark mode) with persistent storage.
    - Partnership Inquiry Form (Formspree).
    - Disqus Comments.
    - Google AdSense Integration (ca-pub-4870116812658624).

## AdSense Approval Optimization (High-Quality Standards)
1.  **Rich Content:** Add an "About" section explaining the mathematics of lotto and how to use the generator to provide value beyond just the tool.
2.  **Trust & Transparency:** Add a **Privacy Policy**, **Terms of Use**, and **Disclaimer** section (mandatory for AdSense).
3.  **Navigation:** Implement a clean navigation system and a footer for easy access to all site sections.
4.  **SEO & Metadata:** Add proper meta descriptions and keywords to help search engines understand the site's value.
5.  **UX Polish:** Ensure fast loading and a perceptually uniform color palette for a premium feel.
